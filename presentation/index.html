SPA - Single Page Applications

A single-page application (SPA) is a web application or web site that fits on a single web page with the goal of providing a user experience similar to that of a desktop application. In an SPA, either all necessary code – HTML, JavaScript, and CSS – is retrieved with a single page load

ReactJS

handlePropsChange 

            handlePropsChange(e) {
              this.props.propsText = "Karol Kowalski";
              console.log("props", this.props.propsText); 
              //logs Karol Kowalski, 
              //but on screen nothing has changed!
            }
            (...)
            
props: {this.props.propsText}

            //logs old value! : "propsText"
Why? We've changed this.props. ReactJS doesnt care about that... :)
handleStateChangeBad 

            handleStateChangeBad(e) {
              //Wrong!! Render function wont be executed! 
              //Direct assign wont work.
              this.state.text = "Karol Kowalski"; 
              
              console.log("state", this.state.text);
              //Logs Karol Kowalski.. :)
            }
            (...)
            
state: {this.state.text} 

            //Mariusz Jasinski
Why? We've changed this.state, but... direct assign to variable wont work.
handleStateChangeGood 

            handleStateChangeGood(e) {
              //Everything is fine!
              //this.setState has been used!
              this.setState({text: "Mariusz Jasinski The Second"}) 
              
              console.log("state", this.state.text);
              //logs Mariusz Jasinski The Second
            }
            (...)
            
state: {this.state.text} 

            //Mariusz Jasinski The Second
Success! We've changed state and also we used this.setState for that!

Conlusion?

Rules to follow

  • this.state has been changed..
  • this.setState has been used (no direct assign allowed).. so
  • this.state.variable = "Mariusz">
  • this.setState({variable: "Mariusz"})

                  variable1: 'testValue'
                });

              }
              render() {}
            }
